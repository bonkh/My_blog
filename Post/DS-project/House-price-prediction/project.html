— PROJECT NAME

House Price Prediction

— CATEGORY

Data Science / Machine Learning

— ROLE

  • Data Cleaning
  • Exploratory Data Analysis (EDA)
  • Model Development
  • Model Evaluation

— TOOLS

Python, pandas, matplotlib, seaborn, scikit-learn, XGBoost

— DATE

04/2025

This project aimed to build a regression model to predict house prices using a dataset containing housing features such as lot size, square footage, number of bedrooms and bathrooms, year built, and neighborhood rating.

After performing thorough data cleaning and exploratory data analysis, various machine learning models were implemented and compared, including linear regression, random forest, and gradient boosting. XGBoost ultimately achieved the best performance in terms of RMSE and R² score.

The project highlights how important proper feature engineering and domain understanding are in producing meaningful models. Future enhancements could include integrating external data sources such as school quality, proximity to public transport, and local amenities.
