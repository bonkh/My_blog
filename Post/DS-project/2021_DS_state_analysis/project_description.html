This is one of my first project about Data Science I finished in the course Programming for Data Science. In this course, we concentrate on building up our programming skills, especially in using data processing and analysis skills with Python and some packages like Numpy or Pandas. In this project, our team's goal is making a detail analysis on a specific dataset.

Our team explored the dataset 2022 Kaggle Machine Learning & Data Science Survey to know more about people who is working with data in different industries and the best ways for new data scientists like us to break into the field.

2022 Kaggle Machine Learning & Data Science Survey is the most comprehensive dataset available on the state of ML and data science. Kaggle Machine Learning & Data Science Survey is a competition held annually from 2017 until 2022 to conduct an industry-wide survey with the aim of presenting a truly comprehensive view of the current state of Data Science and Machine Learning. The survey took place from 09/16/2022 to 10/16/2022, after Kaggle cleaned the data (Data Cleaning), it received 23,997 responses.

The questions in this survey contains many topics here:

  • Biography: capture personal and demographic information about the participant, such as age, gender, location, and student status.
  • Education: explore the paricipant learning journey in Data Science: where they studied, which resources were most useful, education level, and research experience.
  • Programming skills & experience: ask about participant’s technical background — coding experience, languages, tools, and familiarity with machine learning.
  • Employments: Describes the participant’s current work situation — role, industry, company size, team structure, ML usage, work activities, and income level
  • Other product used (Cloud computing, Data storage...): Identifies which cloud, data storage, and related tools the participant uses and how much they have invested in them.

Through the project, each memember pick a topic and deeply dive into it. In my section, I nvestigated key trends about the data science state among industries, salary ranges and required skills for various data roles and programming language usage patterns.
